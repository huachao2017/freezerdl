3、询问训练进度接口(get)：http://$dns$/api/trainstatus?groupid=xx&modelid=xx
返回200：
{
    "status": 0,                          -- 0:不存在，1:排队中，2：正在训练，3：训练结束
    "complete_percent": 0.22,             -- 完成比例
    "calculate_time": 137,                -- 如训练结束，则训练时间
    "precision": 0.98,                    -- 如训练结束，则返回模型精准度
}
